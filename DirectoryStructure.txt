## Building and Installation Process:

1. Build the project:
   - Run build.bat to compile both the service and setup tool
   
2. Install the service:
   - Run install.bat or install.ps1 as Administrator
   - Follow the interactive setup wizard
   - Service will be installed and started automatically

## Key Components:

- ProductWatcherWorker: The main background service that continuously monitors the database
- ProductDataService: Handles database connections and queries with incremental updates
- ApiService: Manages sending data to your API endpoint
- Setup Console App: Interactive configuration wizard during installation
- Installation Scripts: Automate the Windows Service installation process

## Configuration Process:

The setup wizard will prompt for:
- Database server IP and port (3011)
- Database name, username, and password
- Connection test validation
- API endpoint URL
- Polling interval settings
